IoT Weather Monitoring Project



I had the opoprtunity to work on this project, for my IoT module during Top-Up. The code was written in C++, using Arduino IDE software to code and upload the program onto the microcontroller.

The IoT Based Weather Monitoring System uses the following electronic sensors : DHT11, MQ-135, LDR and rain sensor to collect and report weather information to the Arduino Nano IoT 33 microcontroller. Current temperature, humidity, air quality, light intensity and rain conditions data from sensors being uploaded to the Arduino IoT Cloud, for monitoring and analysis.


I used Fritzing Software to design the wiring diagram for the project, as can be seen above.

Fritzing is an excellent open-source platform for teaching, sharing, and prototyping electronic creations. It allowed me to develop components, which I incorporated to the wiring diagram.

Monitoring and observing various weather conditions has been the focus of weather monitoring innovations. The project uses Arduino IoT Cloud application to receive data from sensors for monitoring and analysis. Arduino IoT Cloud facilitates rapid, simple, and secure construction of connected things by users. Arduino IoT Cloud is a web-based platform that facilitates the creation, deployment, and monitoring of IoT projects.
